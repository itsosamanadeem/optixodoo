Purchase Analytic Distribution

This module enable you to set analytic distribution on Purchase header and it also enable you to update lines analytic account with the header Purchase.

Key Features

  • You can set analytic distribution on Purchase Order.
  • Automatically set the analytic distribution to the line analytic distribution Once created a new line.
  • Calibre to update lines analytic distribution to Account Move analytic distribution using single click.
  • Account Move analytic distribution will be visible only for users who have "Analytic Accounting" right.
